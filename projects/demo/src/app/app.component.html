Pipe

Use: 
"{{ '{{' }} list | oxford:'and' }}"
Output:
  • {{ list.length }}: "{{ list | oxford:'and' }}"
Truncate a list: 
  "{{ '{{' }} list | oxford: 'and' : {{ '{' }} trail: 'etc', wordCount: 5 {{ '}' }} }}"
Output:
{{ longList | oxford : 'and' : {trail: 'etc', wordCount: 5} }}

Component

Use: 
"<ng-container *ngFor="let x of list; let i = index">
  <a [href]="'https://www.google.com/search?q=' + x">{{ '{{' }} x }}</a>
  <ngx-oxford [index]="i" [length]="list.length">or</ngx-oxford>
</ng-container>"
Output:
  • {{ list.length }}: " {{ x }} or "
